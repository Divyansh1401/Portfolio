# Kinko Case Study Images

Used in: `index.html` → Kinko case study overlay (`data-project="kinko"`)

## Naming convention
```
thumbnail.jpg        ← homepage project card thumbnail (P1 row)
cover.jpg            ← case study hero banner
screen-*.jpg         ← app screens / design artifacts
```

## Specs
| Image | Aspect ratio | Min size |
|-------|-------------|----------|
| thumbnail.jpg | 16:9 or custom | 800 × 500 px |
| cover.jpg | ~21:9 wide | 1400 × 600 px |
| screen-*.jpg | varies | 800 px wide min |

## Current state
Kinko case study content is deferred — the overlay shell exists but content is placeholder.
Once images are ready, update the `caseStudies.kinko` JS object in `index.html`.
